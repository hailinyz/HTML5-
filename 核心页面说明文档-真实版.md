## 2.1 核心页面

本章节对网站的核心页面进行详细说明，展示各页面的功能模块和技术实现。网站共包含1个主页和4个二级页面，满足HTML5程序设计期末大作业的要求。

### 2.1.1 首页 (Home.vue)

**页面截图：**
> [请在此处插入首页截图]

**页面说明：**

首页是网站的核心入口页面，采用响应式布局设计，包含以下主要功能模块：

**① 页头导航栏**
- 位置：页面顶部
- 功能：网站Logo、导航菜单（首页/文章/视频/关于/联系）、主题切换按钮、搜索按钮
- 技术实现：使用Element Plus的`<el-header>`和`<el-menu>`组件
- 响应式：桌面端显示水平菜单，移动端（768px以下）切换为抽屉式菜单
- **开发说明**：导航栏基础结构参考Element Plus官方文档示例，响应式菜单切换逻辑使用AI辅助生成

**② 公告栏**
- 位置：页头下方
- 功能：展示网站最新消息和重要通知
- 技术实现：使用Element Plus的`<el-alert>`组件
- **开发说明**：组件使用参考Element Plus文档，样式调整为自己编写

**③ 轮播图区域**
- 位置：页面左侧主要区域
- 功能：自动轮播展示网站重点内容，支持左右箭头切换
- 技术实现：使用Naive UI的`<n-carousel>`组件，设置自动播放间隔4秒
- **开发说明**：轮播图组件直接使用Naive UI库，配置参数参考官方文档

**④ 分类卡片**
- 位置：轮播图右侧
- 功能：展示"编程"、"人生"、"生活"三个分类入口
- 技术实现：使用CSS3渐变背景和过渡效果
- **开发说明**：卡片样式参考网上教程，渐变色搭配为自己调整

**⑤ 文章列表区域**
- 位置：页面中部主要内容区
- 功能：展示博客文章卡片，支持标签筛选和分页
- 技术实现：使用Element Plus栅格系统实现响应式布局
- **开发说明**：文章卡片布局参考Element Plus官方示例，数据渲染逻辑使用AI辅助编写

**⑥ 侧边栏 - 个人信息卡片**
- 位置：页面右侧
- 功能：展示个人头像、姓名、座右铭、社交链接
- 技术实现：使用CSS3渐变背景和动画效果
- **开发说明**：
  - 卡片基础样式使用Element Plus的`<el-card>`组件
  - **渐变背景和动画效果引用自UIverse.io网站**（Cards组件库）
  - 个人根据需求调整了颜色和尺寸

**⑦ 侧边栏 - 翻转卡片**
- 位置：个人信息卡片下方
- 功能：鼠标悬停时卡片翻转，展示不同内容
- 技术实现：使用CSS3的`transform: rotateY(180deg)`实现3D翻转效果
- **开发说明**：
  - **翻转卡片效果引用自UIverse.io网站**（Cards组件库）
  - 图片和内容为自己替换

**⑧ 侧边栏 - 统计卡片**
- 位置：翻转卡片下方
- 功能：展示博文数、分类数、访客数统计
- 技术实现：使用`<el-card>`组件，配合图标展示
- **开发说明**：卡片布局为自己编写，图标使用Element Plus Icons

**⑨ 页脚**
- 位置：页面底部
- 功能：展示版权信息和工作室名称
- 技术实现：使用`<el-footer>`组件
- **开发说明**：简单的文字布局，自己编写

**技术要点：**
- HTML5：语义化标签、响应式图片
- CSS3：Flexbox布局、Grid布局、动画效果、渐变背景、圆角边框
- JavaScript：Vue 3组合式API、响应式数据、路由跳转

### 2.1.2 视频列表页 (Videos.vue)

**页面截图：**
> [请在此处插入视频列表页截图]

**页面说明：**

视频列表页是网站的多媒体展示核心页面，集成了HTML5视频和音频播放功能，满足大作业对多媒体控制的要求。

**① 页面标题**
- 位置：页面顶部
- 功能：显示"视频教程"标题
- **开发说明**：简单的标题样式，自己编写

**② 视频播放区域**
- 位置：页面主体上部
- 功能：展示3个教程视频，每个视频配有完整的播放控制功能
- 技术实现：
  - 使用HTML5原生`<video>`标签实现视频播放
  - 视频尺寸：宽度850px，高度400px
  - 响应式适配：移动端自动调整为100%宽度
- **开发说明**：
  - `<video>`标签使用参考MDN文档和课堂教学内容
  - 视频文件路径配置为自己设置

**③ 视频控制面板**
- 位置：每个视频右侧
- 功能：提供完整的视频播放控制
- 技术实现（使用原生JavaScript Video API）：
  - 播放/暂停：`video.play()` / `video.pause()`
  - 快退/快进：`video.currentTime += value`
  - 播放速度：`video.playbackRate = 1/1.5/2`
  - 音量调节：`video.volume += 0.1`
  - 静音控制：`video.muted = !video.muted`
  - 时间显示：实时显示当前时间/总时长
- **开发说明**：
  - Video API的基本用法参考课堂教学和MDN文档
  - **控制按钮的布局和样式使用AI辅助生成**
  - **时间格式化函数（formatTime）使用AI辅助编写**
  - 按钮点击事件处理逻辑为自己编写

**④ 视频信息展示**
- 位置：视频下方
- 功能：显示视频标题、描述、时长、观看次数
- **开发说明**：简单的文字和图标布局，参考Element Plus文档

**⑤ 音频播客区域**
- 位置：页面下部
- 功能：提供音频播放功能，支持歌曲切换和进度控制
- 技术实现：
  - 使用HTML5原生`<audio>`标签实现音频播放
  - 进度条：可视化显示播放进度，支持点击跳转
  - 播放控制：上一曲、播放/暂停、下一曲、随机播放
- **开发说明**：
  - `<audio>`标签使用参考MDN文档和课堂教学内容
  - **进度条样式和交互逻辑使用AI辅助生成**
  - 歌曲切换逻辑为自己编写
  - 播放控制按钮样式参考网上教程

**技术要点：**
- **HTML5多媒体特性**：
  - `<video>`标签（课堂学习内容）
  - `<audio>`标签（课堂学习内容）
  - Video API（参考MDN文档）

## 2.4 响应式设计实现

所有核心页面均采用响应式设计，确保在不同设备上都有良好的显示效果。

**断点设置：** 768px（课程要求）

**响应式技术实现：**

**① Element Plus栅格系统**
```vue
<el-col :xs="24" :md="8">
  <!-- 移动端占满24格，桌面端占8格 -->
</el-col>
```
- **开发说明**：栅格系统使用参考Element Plus官方文档

**② CSS媒体查询**
```css
@media (max-width: 768px) {
  .desktop-menu { display: none; }
  .mobile-menu-icon { display: block; }
}
```
- **开发说明**：媒体查询语法参考课堂教学内容和MDN文档

**③ 移动端菜单**
- 桌面端：水平导航菜单
- 移动端：抽屉式菜单
- **开发说明**：抽屉菜单使用Element Plus的`<el-drawer>`组件，参考官方文档
