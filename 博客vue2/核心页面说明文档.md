本章节对网站的核心页面进行详细说明，展示各页面的功能模块和技术实现。网站共包含1个主页和4个二级页面，满足HTML5程序设计期末大作业的要求。

### 2.1.1 首页 (Home.vue)

**页面截图：**
> [请在此处插入首页截图]

**页面说明：**

首页是网站的核心入口页面，采用响应式布局设计，包含以下主要功能模块：

**① 页头导航栏**
- 位置：页面顶部
- 功能：网站Logo、导航菜单（首页/文章/视频/关于/联系）、主题切换按钮、搜索按钮
- 技术实现：使用Element Plus的`<el-header>`和`<el-menu>`组件，配合CSS3过渡效果实现滚动吸顶
- 响应式：桌面端显示水平菜单，移动端（768px以下）切换为抽屉式菜单

**② 公告栏**
- 位置：页头下方
- 功能：展示网站最新消息和重要通知
- 技术实现：使用`<el-alert>`组件，配合CSS3的`slideDown`动画效果

**③ 轮播图区域**
- 位置：页面左侧主要区域
- 功能：自动轮播展示网站重点内容，支持左右箭头切换
- 技术实现：使用Naive UI的`<n-carousel>`组件，设置自动播放间隔4秒
- HTML5特性：使用`<img>`标签加载图片资源

**④ 分类卡片**
- 位置：轮播图右侧
- 功能：展示"编程"、"人生"、"生活"三个分类入口
- 技术实现：使用CSS3渐变背景（linear-gradient）和过渡效果
- 动画效果：悬停时卡片向右平移10px（transform: translateX）

**⑤ 文章列表区域**
- 位置：页面中部主要内容区
- 功能：展示博客文章卡片，支持标签筛选和分页
- 技术实现：
  - 使用Element Plus栅格系统（`<el-row>`、`<el-col>`）实现响应式布局
  - 桌面端：2列布局（`:md="12"`）
  - 移动端：1列布局（`:xs="24"`）
- 交互效果：悬停时文章封面图放大1.1倍（transform: scale）

**⑥ 侧边栏**
- 位置：页面右侧
- 功能：展示个人信息卡片、翻转卡片、统计数据
- 技术实现：
  - 个人信息卡片：使用CSS3渐变背景和旋转动画
  - 翻转卡片：使用CSS3的`transform: rotateY(180deg)`实现3D翻转效果
  - 统计卡片：使用`<el-card>`组件展示博文数、分类数、访客数

**⑦ 页脚**
- 位置：页面底部
- 功能：展示版权信息和工作室名称
- 技术实现：使用`<el-footer>`组件，固定高度布局

**技术要点：**
- HTML5：语义化标签、响应式图片
- CSS3：Flexbox布局、Grid布局、动画效果（fadeIn、slideDown）、渐变背景、圆角边框（border-radius: 20px）
- JavaScript：Vue 3组合式API、响应式数据（ref、reactive）、路由跳转

### 2.1.3 视频列表页 (Videos.vue)

**页面截图：**
> [请在此处插入视频列表页截图]

**页面说明：**

视频列表页是网站的多媒体展示核心页面，集成了HTML5视频和音频播放功能，满足大作业对多媒体控制的要求。

**① 页面标题**
- 位置：页面顶部
- 功能：显示"视频教程"标题
- 技术实现：使用`<h1>`标签，居中显示

**② 视频播放区域**
- 位置：页面主体上部
- 功能：展示3个教程视频，每个视频配有完整的播放控制功能
- 技术实现：
  - **HTML5 Video标签**：`<video>`标签实现视频播放
  - 视频尺寸：宽度850px，高度400px
  - 响应式适配：移动端自动调整为100%宽度，保持16:9比例

**③ 视频控制面板**
- 位置：每个视频右侧
- 功能：提供完整的视频播放控制
- 技术实现（使用原生JavaScript Video API）：
  - **播放/暂停**：`video.play()` / `video.pause()`
  - **快退5秒**：`video.currentTime -= 5`
  - **快进5秒**：`video.currentTime += 5`
  - **播放速度控制**：`video.playbackRate = 1/1.5/2`
  - **音量调节**：`video.volume += 0.1` / `video.volume -= 0.1`
  - **静音控制**：`video.muted = !video.muted`
  - **时间显示**：实时显示当前时间/总时长（格式：mm:ss）
- 按钮样式：使用CSS3渐变背景（紫色主题），悬停时颜色加深并上移2px

**④ 视频信息展示**
- 位置：视频下方
- 功能：显示视频标题、描述、时长、观看次数
- 技术实现：使用`<el-icon>`组件展示图标，配合文字说明

**⑤ 音频播客区域**
- 位置：页面下部
- 功能：提供音频播放功能，支持歌曲切换和进度控制
- 技术实现：
  - **HTML5 Audio标签**：`<audio>`标签实现音频播放
  - 当前歌曲信息显示
  - **进度条控制**：
    - 可视化进度条（宽度百分比显示播放进度）
    - 点击进度条跳转到指定位置
    - 实时显示当前时间和总时长
  - **播放控制按钮**：
    - 上一曲：切换到上一首歌曲
    - 播放/暂停：切换播放状态
    - 下一曲：切换到下一首歌曲
    - 随机播放：随机选择歌曲播放
- 按钮样式：不同颜色区分功能（红色、蓝色、绿色）

**技术要点：**
- **HTML5多媒体特性**：
  - `<video>`标签及其属性（src、width、height）
  - `<audio>`标签及其属性
  - Video API（play、pause、currentTime、duration、playbackRate、volume、muted）
  - Audio API（play、pause、currentTime、duration）
  - 事件监听（timeupdate、loadedmetadata、ended）
- **CSS3**：
  - 卡片布局、渐变按钮、进度条样式
  - 悬停效果（transform: scale(1.05)）
- **JavaScript**：
  - DOM操作（getElementById、querySelector）
  - 事件处理（@click、@timeupdate）
  - 时间格式化函数（formatTime）
  - 响应式数据管理（ref、reactive）

- **姓名输入框**（`<el-input>`）
  - 验证规则：
    - 必填验证：`required: true`
    - 长度验证：`min: 2, max: 20`
    - 触发时机：`trigger: 'blur'`（失去焦点时验证）
  - 错误提示：红色文字显示错误信息

- **邮箱输入框**（`<el-input>`）
  - 验证规则：
    - 必填验证：`required: true`
    - 格式验证：`type: 'email'`（使用正则表达式验证邮箱格式）
    - 触发时机：`trigger: 'blur'`
  - 错误提示："请输入有效的邮箱地址"

- **电话输入框**（`<el-input>`）
  - 验证规则：
    - 可选字段（非必填）
    - 自定义验证器：`validator: validatePhone`
    - 正则表达式：`/^1[3-9]\d{9}$/`（验证11位手机号）
    - 触发时机：`trigger: 'blur'`
  - 错误提示："请输入有效的手机号码"

- **主题下拉选择**（`<el-select>`）
  - 选项：合作咨询、技术交流、意见反馈、其他
  - 验证规则：
    - 必填验证：`required: true`
    - 触发时机：`trigger: 'change'`（选择改变时验证）
  - 错误提示："请选择留言主题"

- **留言内容文本域**（`<el-input type="textarea"`）
  - 行数：6行
  - 验证规则：
    - 必填验证：`required: true`
    - 长度验证：`min: 10, max: 500`
    - 触发时机：`trigger: 'blur'`
  - 错误提示："留言内容长度在 10 到 500 个字符"

**⑤ 提交按钮**
- 位置：表单底部
- 功能：提交表单数据，触发完整验证
- 技术实现：
  - 使用`<el-button type="primary">`
  - 点击时调用`submitForm()`方法
  - 加载状态：`:loading="loading"`（提交时显示加载动画）
- **JavaScript表单验证流程**：
  ```javascript
  const submitForm = async () => {
    if (!formRef.value) return
    
    // 执行表单验证
    await formRef.value.validate((valid) => {
      if (valid) {
        loading.value = true
        // 模拟提交（实际项目中调用API）
        setTimeout(() => {
          loading.value = false
          ElMessage.success('留言发送成功！')
          formRef.value.resetFields()  // 重置表单
        }, 1500)
      } else {
        ElMessage.error('请检查表单填写是否正确')
      }
    })
  }
  ```

**⑥ 验证反馈**
- 实时验证：输入框失去焦点时自动验证
- 错误提示：红色边框 + 红色错误文字
- 成功提示：绿色消息提示框（ElMessage.success）
- 失败提示：红色消息提示框（ElMessage.error）

**技术要点：**

## 2.2 响应式设计实现

所有核心页面均采用响应式设计，确保在不同设备上都有良好的显示效果。

**断点设置：**
- 桌面端：768px以上
- 移动端：768px以下

**响应式技术实现：**

**① Element Plus栅格系统**
```vue
<el-col :xs="24" :md="8">
  <!-- 移动端占满24格，桌面端占8格 -->
</el-col>
```

**② CSS媒体查询**
```css
/* 移动端适配 */
@media (max-width: 768px) {
  .desktop-menu {
    display: none;  /* 隐藏桌面端菜单 */
  }
  
  .mobile-menu-icon {
    display: block;  /* 显示移动端菜单图标 */
  }
  
  .main-content {
    padding: 20px 15px;  /* 减小内边距 */
  }
}
```

**③ 移动端菜单**
- 桌面端：水平导航菜单（`<el-menu mode="horizontal">`）
- 移动端：抽屉式菜单（`<el-drawer>`）
- 切换触发：点击汉堡菜单图标

**响应式效果：**
- 布局自动调整：多列变单列
- 字体大小缩放：标题字号减小
- 图片自适应：宽度100%，高度自动
- 按钮尺寸调整：移动端按钮更大，便于触摸
